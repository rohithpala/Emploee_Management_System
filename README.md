Objective:
The main objective of this project is to manage the employee records in an organization having
a-number-of branches. Signup, Login features are used in this project. Admins of the
organization can use their credentials and can login. If he is a new admin, he can also signup by
giving a username and password. Every username is unique. Managing includes adding, deleting,
modifying the employee records. Also when hikes are declared, employee salaries are modified.
Based on their Working hours reward points are also given.

Description:
1) At First the program asks the user to Sign-up or Login.
2) While Signing up, if the username is already present it prints a message saying – “Admin
with given Username already exists”.
3) While Logging in, if the credentials given are wrong then the program prints a message
saying – “No Admin with given Credentials”.
4) After successful signup or login the user will be prompted to give the head-quarters
name.
5) Then the user enters the details of employees at that branch
6) The above 2 steps are performed in a similar manner for branches too.
7) After Completely entering the details of employees in each branch, the linked list in
which these details are added is added as a node of a tree.
8) Then the user will be prompted with a message if they want to check or modify the
details of the employees at a particular branch.
9) If yes, a particular function will be called and the details can be modified
10) If no, the program terminates.
